# 写一些代码
通过手写一些代码, 去了解常见功能的基本原理, 顺便踩一下坑以增强记忆.
主要目的:
1. 通过手写一些代码, 加强对原理的理解, 
2. 有些坑只有踩一下才会加强记忆
3. 把他们写出来整理好, 方便以后查阅
4. 存放一些常用的工具代码, 以后拿来即用, 减少重复实现.
